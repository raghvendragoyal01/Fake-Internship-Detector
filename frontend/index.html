Developer API Portal

Integrate ScamShield's ML detection engine directly into your ATS or job board.

Endpoint: POST /api/v1/analyze

Analyze a job posting programmatically.

curl -X POST https://api.scamshield.io/v1/analyze \
  -H "Authorization: Bearer ********************************" \
  -H "Content-Type: application/json" \
  -d '{
    "job_description": "We are looking for a remote data entry clerk...",
    "job_url": "https://suspicious-domain.com/job",
    "recruiter_email": "hr@suspicious-domain.com"
  }'
{
  "success": true,
  "data": {
    "scam_score": 92,
    "risk_level": "HIGH",
    "domain_age_days": 14,
    "suspicious_keywords": ["wire transfer", "advance fee", "telegram"],
    "recommendation": "DO NOT APPLY. High probability of advance-fee fraud."
  }
}
